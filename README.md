## Getting Started

1. Install dependencies...
```bash
npm install
```
2. Start local MySQL db
```bash
cd docker/
docker compose up -d
```
3. Start dev server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
